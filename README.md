# Frontend Exercises and Projects

A collection of frontend exercises, challenges, and projects designed to practice and demonstrate skills in HTML, CSS, JavaScript, frameworks, and related tooling.

## Contents

- **Exercises**: Small focused tasks for learning or improving specific frontend concepts.
- **Projects**: More complete applications or UI implementations that combine multiple skills.
- **Resources**: Useful links, tools, and references used during development.

## Structure

Each exercise or project should include:

- `README.md` or brief description of the task
- Source files for HTML, CSS, JavaScript, and assets
- Notes on features, implementation details, and learning outcomes

## How to Use

1. Browse the folders to find an exercise or project.
2. Open the source files in your browser or code editor.
3. Run locally using a static server, framework CLI, or live preview.
4. Review the notes and add your own improvements or variations.

## Suggested Exercises

- Responsive layout practice
- Component styling and UI design
- DOM manipulation and event handling
- API fetching and async data rendering
- Form validation and user interactions

## Suggested Projects

- Portfolio website
- Interactive dashboard
- Todo list or task manager
- Weather app with external API
- E-commerce product listing

## Notes

Use this repository as a workspace for frontend learning and experimentation. Keep the README updated with new exercises, instructions, and technologies as the collection grows.
